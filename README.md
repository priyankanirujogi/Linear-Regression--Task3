# Linear Regression Task 3

## Objective
Implement and understand simple & multiple linear regression.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab

## Dataset Used
California Housing Dataset (built-in from Scikit-learn)

## Steps Performed
1. Imported libraries
2. Loaded dataset
3. Split data into train and test sets
4. Trained Linear Regression model
5. Predicted outputs
6. Evaluated using MAE, MSE, R²
7. Visualized Actual vs Predicted Prices
8. Interpreted coefficients
